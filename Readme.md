# react-sso-demo
A demo of react sso.

# Local Development

1. Install node_modules
```
npm run install
```
or
```
yarn
```
2. Start server
```
npm run dev
```


# Prod
1. Local test prod env
```
npm run build
npm run start
```
2. Deploy to prod or test env
You can deploy dist directory to server, then execute `node index.js` in dist directory.
